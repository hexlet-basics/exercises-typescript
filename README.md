## How to contribute

* Discuss the project on Telegram: https://t.me/hexletcommunity/12

### Requirements

* docker
* docker compose V2
* make

## Develop

```bash
# setup
make
# run
make compose
# check
make ci-check

# run tests
make compose-test

# run linters and validators
make code-lint
make compose-description-lint
make compose-schema-validate
```
